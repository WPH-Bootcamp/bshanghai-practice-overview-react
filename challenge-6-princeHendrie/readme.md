# Challenge 6 - Transform To Do App using TypeScript

# Description

In this challenge, you'll build a command-line To-Do List application using TypeScript. This challenge is designed to reinforce your understanding of core JavaScript concepts including arrays, objects, functions, control flow, and user input handling.

You will be given a basic project structure with function stubs, and your task will be to implement the logic to create a fully functional to-do list manager.

# Folder Structure

You are provided with the following folder structure:

```
root/
├── index.ts
├── readme.md
├── package.json
├── package-lock.json
└── .gitignore
```

You will only need to modify the index.ts file for this assignment.

# Assignment Objectives

## 1. The code should have no error because of type

### Use Everything you know

- give type
- handle import
- type guard

## 2. Expected Output Examples

### Adding a Todo:

[ACTIVE] | Buy groceries

[ACTIVE] | Do laundry
Enter the NUMBER of the to-do to mark as completed: 1
To-do "Buy groceries" marked as completed.

### Deleting:

[DONE] | Buy groceries

[ACTIVE] | Do laundry
Enter the NUMBER of the to-do to delete: 2
To-do "Do laundry" deleted.

### Listing Todos:

--- YOUR TO-DO LIST ---
[DONE] | Buy groceries
[ACTIVE] | Study JavaScript

### 4. Getting Started

Install dependencies:

```
npm install
```

To run your to-do list application:

```
npm run dev
```
